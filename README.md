# Assessment Recommendation Engine

This project implements a simple assessment recommendation engine using a rule-based approach. 
It recommends suitable assessments for a candidate based on role, skills, and experience level.

## Files
- recommendation_engine.py: Core recommendation logic
- assessments.json: Sample SHL assessment catalogue
- requirements.txt: Project dependencies

## How to Run
1. Clone the repository
2. Run: `python recommendation_engine.py`

## Output
The script prints a list of recommended assessments for a sample candidate profile.
